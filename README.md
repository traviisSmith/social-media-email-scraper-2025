## Introduction

Social Media Email Scraper 2025 is an API designed for retrieving verified emails from major social platforms using user IDs. This tool is ideal for developers building apps that need to gather contact information for marketing, influencer outreach, or market research.

### Why Choose Social Media Email Scraper 2025?

- **Cross-Platform Lookup:** Query emails via User ID from YouTube, TikTok, LinkedIn, and more.
- **Real-Time Validation:** Get verified email data with millisecond response times.
- **Scalable & Secure:** Built with FastAPI for fast performance and rate-limited for high-traffic applications.

## What Data This Scraper Extracts

| Field Name   | Field Description                                               |
|--------------|------------------------------------------------------------------|
| `platform`   | The social platform from which the email was retrieved.          |
| `userId`     | The unique ID of the user on the platform.                       |
| `email`      | The verified email address associated with the user ID.          |
| `timestamp`  | The timestamp when the data was retrieved.                       |
| `isVerified` | A boolean indicating whether the email address is verified.      |
